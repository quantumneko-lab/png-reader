# 電子書籍ツール - E-Book Viewer

PNG画像を電子書籍として管理・閲覧できるデスクトップアプリケーションです。

## 機能

### 1. 書籍管理
- **新規作成**: 新しい書籍プロジェクトを作成
- **保存/読み込み**: JSONフォーマットで書籍データを永続化
- **複数書籍管理**: 複数の書籍を管理・切り替え可能

### 2. 画像管理
- **複数ファイルアップロード**: PNG形式の画像ファイルをバッチ追加
- **ページサムネイル**: ページの一覧表示とサムネイル表示
- **画像ビューア**: フルスクリーン表示に対応した画像ビューア

### 3. ページ操作
- **ページ順序変更**: ↑↓ボタンまたはコンテキストメニューでページ順序を変更
- **ページ削除**: 不要なページを削除
- **表紙設定**: 任意のページを表紙として設定

## インストール

### 前提条件
- Python 3.7以上

### セットアップ

1. 仮想環境の作成（推奨）:
```bash
python -m venv venv
```

2. 仮想環境の有効化:
- Windows:
```bash
venv\Scripts\activate
```
- macOS/Linux:
```bash
source venv/bin/activate
```

3. 依存パッケージのインストール:
```bash
pip install -r requirements.txt
```

## 実行方法

```bash
python main.py
```

## 使用方法

### 1. 新しい書籍を作成
- 「新規作成」ボタンをクリック
- 新しい書籍プロジェクトが作成されます

### 2. 画像をアップロード
- 「画像を追加」ボタンをクリック
- PNG形式の画像ファイルを複数選択
- 選択した画像がページとして追加されます

### 3. ページを管理
- 左パネルのページリストでページを選択
- 右パネルで選択したページが表示されます
- ↑↓ボタンでページ順序を変更
- 「削除」で不要なページを削除
- 「表紙に設定」で表紙を指定

### 4. 書籍を保存
- 「保存」ボタンをクリック
- ファイル名を入力して保存
- JSONフォーマットで保存されます

### 5. 書籍を開く
- 「開く」ボタンをクリック
- 保存されている書籍一覧から選択
- 「開く」で書籍を読み込み

## 依存パッケージ

- **PyQt5**: デスクトップUI構築フレームワーク
- **Pillow**: 画像処理ライブラリ
